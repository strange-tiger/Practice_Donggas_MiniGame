C# 에서 중간 언어의 이름은 무엇인가?
- IL(Intermediate Language)

리터럴이란 무엇인가? 예시를 들며 설명하라
- 리터럴은 고정값을 나타내는 표기법으로, 예를 들어 int a = 10; 의 코드에서 a는 변수 및 변수의 식별자, 10은 리터럴이다.

값 타입과 참조타입의 차이를 설명하라
- 값 타입은 스택 메모리 저장, 가비지 컬렉션 대상 외
- 참조 타입은 힙 메모리 저장, 가비지 컬렉션 대상

2진수로 10을, 16진수로 30을 표현해보아라.
- 10 -> 1010(2)
- 30 -> 1E(16)

숫자의 양수와 음수를 비트로 어떻게 표기하여 구분하는가?
- 정수값의 맨 앞 비트인 부호 비트가 0일 때 양수, 1일 때 음수로 구분한다. 

int 타입에 uint타입으로 담을 수 있는 가장 큰 값을 형변환하여 넣으면 -1이 나오게 된다. 이처럼 담을 수 있는 범위를 초과하는 값을 집어 넣었을때 엉뚱한 값이 나오는 현상을 무엇이라고 부르는가?
- 오버플로우

var 변수는 무엇인가? var 변수는 언제 타입이 정해지는가?
- 컴파일러가 그 형식을 정하는 변수로, 컴파일 때 데이터를 보고 타입이 정해진다.

정수형 타입인 값 10을 왼쪽으로 1칸 시프트 연산을 하면 값이 어떻게 되는가?
- 2배가 되어 20이 된다.

멤버변수는 무엇인가?
- 클래스가 갖는 변수, 클래스 내에 선언된 변수 요소.

매개변수는 무엇인가?
- 메소드(함수)에 인자로서 입력되는 변수

ref, in, out 키워드의 차이는 무엇인가?
- ref 키워드로 한정된 매개변수 및 반환값을 참조 타입으로 전달한다.
- in 키워드로 한정된 매개변수는 입력 전용 매개변수로, 참조로 전달된 매개변수의 변경, 저장을 허용하지 않는다.
- out 키워드로 한정된 매개변수는 출력 전용 매개변수로, 메소드가 전달 받은 매개변수에 값을 저장할 것을 강제한다. 

is와 as는 무엇인가?
- 형식 변환을 위한 연산자로, is는 변수가 해당 형식인지를 검사해 결과를 bool값으로 반환하고 as는 해당 형식으로 변수를 변환하는데, as가 실패하면 변수는 null 값을 가진다.
